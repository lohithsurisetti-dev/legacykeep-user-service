# user-service

## Description
This is the user-service microservice for LegacyKeep.

## Technology Stack
- Spring Boot 3.x
- Java 17
- Maven

## Quick Start
```bash
mvn spring-boot:run
```

## API Documentation
Swagger UI: http://localhost:8080/swagger-ui.html

## Health Check
Health endpoint: http://localhost:8080/actuator/health
